## Analytics Dashboard Structure

**Focal Point:** RadarChart (Skill Map) — Primary visual anchor, positioned at top of dashboard after basic metrics

### Metric Group 1: Basic & Engagement Metrics (Cards)
- Score trends: Recent assignment scores with direction indicator (up/down)
- On-time rate: Percentage with icon
- Time spent: Total time in minutes/hours

### Metric Group 2: Skill Map (Radar Chart) — Core Feature
- RadarChart from fl_chart
- Displays mastery_level (0.0 - 1.0) for each skill
- Skills from student_skill_mastery + learning_objectives

### Metric Group 3: Deep Strength/Weakness Analysis (Bar + Table)
- Accuracy by tag: BarChart showing correct rate per topic
- Difficulty vs score: Identify if student performs on hard/easy questions
- Time per question: Highlight slow questions

### Metric Group 4: Context & Trajectory (Cards + Line)
- Class comparison: Anonymous class average (bypasses RLS via view)
- Trajectory: LineChart predicting future performance

## Empty States (Contextual)

| Condition | Trigger | UI |
|-----------|---------|-----|
| ZERO SUBMISSIONS | submissions count = 0 | Welcome message + "Take Diagnostic Test" CTA |
| PENDING GRADING | ai_graded = false OR total_score IS NULL | Skeleton loader / "AI is analyzing..." |
| NO SKILL / PARTIAL | student_skill_mastery has attempts = 0 OR metrics missing | Charts grayed out + "Not enough data" card |
| NORMAL | All tables have data | Full hybrid dashboard |

**Critical:** Empty state containers MUST match actual chart heights (300px for radar) to prevent layout shift.

## fl_chart Configuration

### RadarChart (Skill Map)
```dart
RadarChart(
  RadarChartData(
    radarBorderData: BorderSide(color: DesignColors.primary),
    gridBorderData: BorderSide(color: DesignColors.dividerLight),
    tickCount: 5,
    ticks: [0, 0.25, 0.5, 0.75, 1.0],
    // Use DesignTypography.caption for labels
  ),
)
```

### LineChart (Grade Trends)
```dart
LineChart(
  LineChartData(
    lineBarsData: [
      LineChartBarData(
        color: DesignColors.primary,
        barWidth: 3,
        belowBarData: BarAreaData(
          show: true,
          color: DesignColors.primary.withValues(alpha: 0.1),
        ),
      ),
    ],
  ),
)
```

### BarChart (Subject Comparison)
```dart
BarChart(
  BarChartData(
    barGroups: [],
    borderData: FlBorderData(show: false),
    gridData: FlGridData(show: true),
  ),
)
```
