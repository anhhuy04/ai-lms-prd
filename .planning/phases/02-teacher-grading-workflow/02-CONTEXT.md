# Phase 2: Teacher Grading Workflow

**Mục tiêu:** Enable teachers to view submissions and grade student work

**Requirements:** TEA-01 to TEA-06

---

## Success Criteria

1. Teacher can view submission list with status filters (ATC model)
2. Teacher can view individual submission details (Side-by-Side)
3. Teacher can grade and provide feedback (Human-in-the-loop)
4. Grade override functional with audit trail
5. Publish grades functionality (Stage Curtain)
6. Quick navigation between submissions
